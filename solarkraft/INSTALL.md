# Solarkraft Runtime Monitoring

## Installing the development version

 1. Install NodeJS 20.x. We recommend using
 [nvm](https://github.com/nvm-sh/nvm).
 
 1. Install dependencies:

    ```sh
    npm install
    ```

 1. Compile:

    ```sh
    npm run compile
    ```

 1. Install a local version:

    ```sh
    npm link
    ```

 1. Run the CLI:

    ```sh
    solarkraft
    ```
